## Install
1. clone the repo   
  
        git clone git://github.com/happypeter/peter-vim.git

1. rename it

        mv peter-vim ~/.vim

1. set up ~/.vimrc
   you need to have a fake .vimrc in your $HOME

        ln -s ~/vim /vimrc .vimrc

1. you may also need to install `Ctags`

        sudo apt-get install exuberant-ctags # for ubuntu
